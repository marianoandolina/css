Responsive Design - Movil First

El diseño responsive es una técnica de diseño web que permite que una página web se adapte a diferentes tamaños de pantalla, desde dispositivos móviles hasta monitores de escritorio. El diseño mobile first es un enfoque del diseño responsive que consiste en diseñar y desarrollar una página web primero para dispositivos móviles y luego adaptarla a dispositivos más grandes. En el diseño responsive tradicional, se empieza diseñando y desarrollando una página web para dispositivos de escritorio y luego se adapta a dispositivos móviles. Esto puede resultar en una experiencia de usuario deficiente en dispositivos móviles, ya que los elementos de la página web pueden estar demasiado grandes o demasiado pequeños. El diseño mobile first resuelve este problema al comenzar con una versión móvil de la página web que es fácil de usar y luego adaptarla a dispositivos más grandes. Esto garantiza que la página web sea accesible y usable para todos los usuarios, independientemente del dispositivo que utilicen.
Para implementar el diseño mobile first, se utilizan las media queries de CSS.
Las media queries permiten especificar reglas de CSS que se aplican a diferentes tamaños de pantalla. Por ejemplo, se puede utilizar una media query para especificar que un elemento se muestre en una sola columna en dispositivos móviles y en dos columnas en dispositivos de escritorio.

Basicamente lo que hay que hacer es crear la pagina e indicarle a los diferentes menus como se van a comportar cuando la resolusion cambie a las diferentes dispositivos.
