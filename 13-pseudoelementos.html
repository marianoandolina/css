Pseudoelementos

La mayoria de los Pseudoelementos no forma parte del dom. Los elementos que requieren la etiqueta content no forman parte del dom. Los pseudoelementos en CSS son una forma de seleccionar y estilizar partes específicas de un elemento HTML. Aquí tienes una breve descripción de los pseudoelementos más comunes: Tenemos que crearlo y aplicarlo sobre un elemento.
::first-line: Este pseudoelemento selecciona la primera línea de un elemento. Se puede utilizar para aplicar estilos especiales a la primera línea de un párrafo, como cambiar su color o aplicar un espaciado personalizado. Solo se aplica a los que no son inline
::first-letter: Este pseudoelemento selecciona la primera letra de un elemento. Se puede utilizar para aplicar estilos especiales a la primera letra de un párrafo, por ejemplo, aumentando su tamaño o cambiando su color. Solo se aplica a los que no son inline
::placeholder: Este pseudoelemento se utiliza para aplicar estilos al texto de marcador de posición en un campo de entrada o área de texto. Se puede utilizar para cambiar el color del texto de marcador de posición o aplicar otros estilos.
::selection: Este pseudoelemento se utiliza para aplicar estilos al texto seleccionado por el usuario. Por ejemplo, se puede cambiar el color de fondo o el color del texto cuando se selecciona.
::before: Este pseudoelemento se utiliza para insertar contenido antes del contenido de un elemento. Se puede utilizar para agregar elementos decorativos o iconos antes del contenido real. Son hijos del elemento al que le estamos aplicando el pseudoelemento y necesitan si o si la propiedad content, funciona tanto en linea como en bloque.
::after: Similar al pseudoelemento ::before, ::after se utiliza para insertar contenido después del contenido de un elemento. Se puede utilizar para agregar elementos decorativos o iconos después del contenido real. Son hijos del elemento al que le estamos aplicando el pseudoelemento y necesitan si o si la propiedad content, funciona tanto en linea como en bloque.
