Overflow

Se refiere al comportamiento de un elemento cuando su contenido excede el espacio disponible para mostrarlo dentro de su contenedor. El atributo CSS "overflow" se utiliza para controlar cómo se gestiona el contenido que desborda los límites de un elemento. Puede aplicarse a elementos como divs, párrafos, imágenes y otros elementos HTML. Existen cuatro posibles valores para la propiedad "overflow":
Visible: Este es el valor predeterminado. Cuando se establece en "visible", el contenido que desborda el elemento se mostrará fuera de los límites del contenedor, sin recortarse ni ocultarse. Puede extenderse más allá de los límites del contenedor y superponerse a otros elementos.
Hidden: Cuando se establece en "hidden", cualquier contenido que desborde el contenedor se ocultará y no se mostrará. Es decir, se recortará y no se verá más allá de los límites del contenedor.
Scroll: Con este valor, se mostrará una barra de desplazamiento en el contenedor, permitiendo al usuario desplazarse para ver el contenido que desborda. Incluso si el contenido no desborda, la barra de desplazamiento estará presente, pero estará deshabilitada.
Auto: Este valor es similar a "scroll" en el sentido de que también muestra una barra de desplazamiento si es necesario. Sin embargo, a diferencia de "scroll", la barra de desplazamiento solo aparecerá si hay contenido que desborde el contenedor.

La propiedad "overflow" es muy útil cuando se trabaja con diseños responsivos y se necesita controlar cómo se muestra el contenido cuando el espacio es limitado. Permite controlar el comportamiento de desbordamiento del contenido y garantizar una experiencia de usuario adecuada.
