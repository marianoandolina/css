Cursor

La propiedad cursor de CSS se utiliza para especificar el tipo de cursor que se mostrará cuando el mouse se encuentre sobre un elemento. Esta propiedad puede ser muy útil para mejorar la experiencia del usuario, ya que puede ayudar a indicar qué acciones se pueden realizar sobre un elemento. Por ejemplo, se puede utilizar la propiedad cursor para mostrar un cursor de mano cuando el mouse se encuentre sobre un enlace, o un cursor de flecha cuando el mouse se encuentre sobre un elemento que se puede arrastrar.

Existen muchas opciones diferentes para la propiedad cursor. Algunas de las opciones más comunes son:

auto: El cursor predeterminado del navegador.
pointer: Un cursor de mano.
move: Un cursor de flecha.
help: Un cursor de interrogación.
wait: Un cursor giratorio.
